# Live Camera Feed Viewer

This project is a simple HTML-based web page that displays a **live camera feed** from your device. The video is centered on a **black background** for a clean and minimalistic look.

## Features

- Small Project on how to display a video in a html webpage.
- Displays the live video feed in the center of the screen.
- Black background for better visual contrast.
- Fully responsive and works on modern browsers.
