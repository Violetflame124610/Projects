# Ultrasonic Radar
This Projects helps in building an ultrasonic radar which teaches fundamental principles of electronics, programming (microcontrollers, serial communication), physics (sound waves, echolocation), and data visualization, bridging hardware with software to create a functional object detection system.

## Materials
- Arduino
- Ultrasonic sensor
- Servo motor
- Breadboard
- Jumper Wires
- USB cable for injecting of code to arduino
- 
## Applicaions we will be using
- Arduino IDE
- Processing 3
